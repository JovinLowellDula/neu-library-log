## Features
- Student, Faculty, and Visitor login with @neu.edu.ph email validation
- Purpose of visit logging (Reading, Research, Computer Use, Group Study, etc.)
- Admin dashboard with visitor statistics by day, week, month, year
- Search by name, student ID, date, or purpose
- Export visitor records to CSV
- Role-based access: Regular users see welcome screen; Admin sees full dashboard
